The Arc
A Systematic Approach to MI

Each project builds on the last. The goal: answer a single question — what computation is happening inside a language model, where, and why does it matter?

🔬

Predict

What does the model output at each layer? LogitLens reads the residual stream at every depth, showing FFN₇ as the dominant contributor to factual token predictions.

🧩

Represent

What does the model know at each layer? LinearProbing shows POS is undecodable from raw embeddings but fully encoded after one transformer block. SparseAutoencoder decomposes features into 2,048 near-orthogonal directions.

⚡

Cause

What is necessary for a prediction? ActivationPatching identifies the exact attention heads responsible for indirect object identification. KnowledgeNeurons finds FFN neurons 47.65× more causally impactful than random.

🎛️

Control

Can we steer behaviour using what we've learned? ActivationSteering uses contrastive vectors; SAEFeatureSteering uses decoder directions — both point to layer 8 as the optimal injection point.

🎯 Cross-Project Convergence Finding

Four independent experiments — using completely different methods — all converge on the same answer: layers 7–8 are GPT-2 Small's semantic hub.

LogitLens FFN₇ has the largest direct logit attribution for factual tokens
ActivationSteering L8 is the optimal CAA injection point; L4 and L11 are significantly weaker
KnowledgeNeurons Peak gradient attribution across all 40 facts is at L7
SAEFeatureSteering L8 hook_resid_pre is where domain features are most causally active

GPT2CircuitAnalysis
Induction head scoring · attention pattern analysis

03 Decompose
SparseAutoencoder
Dictionary learning · L1 sparsity · feature geometry
Train a 2,048-feature ReLU SAE on GPT-2 Small's layer-8 residual stream. 0 dead features after training; feature directions are near-orthogonal. Precursor to SAEFeatureSteering.

04 Theory
ToyModelsOfSuperposition
Superposition · sparsity phase transitions · geometry
Replicate Elhage et al. (2022). Show that effective features exceed hidden dimensions at high sparsity — demonstrating that superposition is a sparsity-driven phenomenon, not a capacity bug.

05 Causal
ActivationPatching
Causal intervention · IOI task · path patching
Run activation patching on the Indirect Object Identification task. L9H9 is the top name-mover head; patching it alone recovers 68% of the clean→corrupt performance gap.

06 Read
LogitLens
Residual stream reading · direct logit attribution
Project every residual stream checkpoint through W_U. FFN₇ contributes the largest direct logit attribution for factual completions; embedding DLA is negative (suppressive).

07 Control
ActivationSteering
CAA · contrastive activation addition · alpha sweep
Derive sentiment steering vectors from paired positive/negative prompts. L8 is the optimal injection point; steering effect is near-linear in α with no abrupt saturation up to α=3.

Grokking · Fourier features · training phase analysis

10 Probing
LinearProbing
Vocabulary-controlled heteronyms · logistic regression
Probe POS at all 13 checkpoints using 6 words that appear in all 3 roles. Raw embedding: 0.139 (below chance). After one block: 0.806. Syntactic encoding happens in a single attention+FFN pass.

11 Formation
InductionHeadFormation
Synthetic sequences · ICL ratio · Olsson et al. replication
Train a 2-layer model on mixed random/repeated sequences. ICL ratio reaches 2.025 (near-optimal) but the canonical look-back attention diagonal doesn't form — showing ICL ≠ induction head circuit at small scale.

12 Causal
KnowledgeNeurons
Gradient × activation attribution · neuron suppression
Attribute 40 factual associations to individual FFN neurons using |grad × act|. Suppressing the top-20 neurons causes a drop 47.65× larger than suppressing 20 random neurons. Peak attribution at L7.

Counter-intuitive: LoRA (0.35% params) forgets MORE than full fine-tuning (+31 vs +5 gen PPL). Update location and LR matter more than parameter count.

17RLHF
DPO
Direct Preference Optimization · frozen ref model · β·log-ratio
Reward gap 0 → 6.12 in 200 steps. Rejected responses pushed 6× further from reference than preferred (−72.5 vs −11.3 log-ratio) — the asymmetry is DPO's core mechanism.
View on GitHub ↗
18RAG
RAG
TF-IDF + dense retrieval · log-prob scoring · MPS batch embed
TF-IDF retrieves correctly 97% of the time, lifting accuracy +18pp. Dense (GPT-2 last-layer mean-pool) only 28% retrieval — motivates sentence-transformers over raw LM embeddings for semantic search.
View on GitHub ↗
19Distil
KnowledgeDistillation
Hinton KD loss · T² scaling · soft vs hard labels
Hard CE outperforms KD at 150 steps (56 vs 149 PPL) with T=4 α=0.7. The T² factor makes KD 16× harder to minimize than CE at short runs — demonstrates why production distillation uses α≈0.5, T≈2, and full training.
View on GitHub ↗honest finding
20Deploy
Quantization
Absmax symmetric PTQ · INT8/INT4 · PPL + latency
INT8: +6.6 PPL with 4× compression (near-lossless). INT4 collapses +3,068 PPL — the 73% quantization error demonstrates exactly why NF4 / per-channel quantization (QLoRA) is needed for 4-bit.
View on GitHub ↗
21RLHF
RewardModeling
Bradley-Terry loss · frozen backbone · scalar head
AUC 0.34 → 0.98 with only 768 trainable parameters (a single linear head on frozen GPT-2). 100% val accuracy at step 100. Shows that preference signal is largely already encoded in the backbone's final hidden state.
View on GitHub ↗
22Merge
ModelMerging
SLERP · domain FT checkpoints · capability frontier
α=0.5 SLERP achieves literary PPL 1.5 AND scientific PPL 1.4 simultaneously — both near-specialist level, zero training. Demonstrates smooth capability interpolation across the full α=[0,1] frontier.
